# Cairo Metro Guide 🚇

A simple mobile application that helps users navigate the Cairo Metro system.  
The app allows users to select their starting station and destination, then displays the full route with all stops between them.

## Features
- Select departure and destination metro stations
- View the full route with all stops
- Track your trip station by station
- Quick access to recent trips
- Simple and clean user interface

## Tech Stack
- Kotlin
- Jetpack Compose
- MVVM Architecture
- State Management

## Use Case
This app is designed for people who use the Cairo Metro and want a simple way to know their route and track their trip easily.
